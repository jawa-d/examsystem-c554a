# 🎓 نظام الامتحانات الذكي

منصة تعليمية متكاملة لإنشاء وإجراء الامتحانات الإلكترونية مع حفظ النتائج في Firebase Firestore.

## 📋 هيكل النظام

### الصفحات الأربع:

#### 1️⃣ **index.html** - الصفحة الرئيسية
- صفحة ترحيب ومعلومات عن النظام
- روابط للانتقال إلى جميع الصفحات
- شرح لخطوات الاستخدام
- قائمة بالمميزات الرئيسية

#### 2️⃣ **questions.html** - إضافة الأسئلة
- إضافة أسئلة جديدة من أنواع مختلفة:
  - ☑️ **اختيار من متعدد** - خيارات متعددة مع إجابة صحيحة واحدة
  - ✔️ **صحيح/خطأ** - سؤال بإجابة صحيح أو خطأ
  - 📝 **إجابة نصية** - سؤال بإجابة نصية قصيرة
- تعيين وقت محدد لكل سؤال (بالثواني)
- حفظ الأسئلة في **Firebase Firestore** تلقائياً
- عرض قائمة الأسئلة المضافة
- حذف أسئلة بشكل فردي أو كلياً

#### 3️⃣ **exam.html** - الامتحان
- **شاشة البداية:**
  - إدخال اسم الطالب (اسم ثلاثي)
  - زر لبدء الامتحان
- **شاشة الامتحان:**
  - عرض الأسئلة واحد تلو الآخر
  - مؤقت ذكي يعد الثواني للسؤال الواحد
  - شريط تقدم يوضح الموقع الحالي
  - نقاط ملاحة للانتقال بين الأسئلة
  - زر للتقدم للسؤال التالي أو إنهاء الامتحان
- **شاشة النتائج:**
  - عرض النسبة المئوية للدرجة
  - إحصائيات (إجابات صحيحة، خاطئة، الوقت المستغرق)
  - تفاصيل لكل سؤال (الإجابة الصحيحة والإجابة الخاطئة)
  - حفظ النتائج في Firebase Firestore تلقائياً
  - خيار إعادة الامتحان

#### 4️⃣ **dashboard.html** - لوحة النتائج
- **الإحصائيات العامة:**
  - عدد الطلاب الذين امتحنوا
  - عدد الامتحانات المأخوذة
  - متوسط النتائج
  - أعلى درجة
- **البحث والتصفية:**
  - البحث عن طالب بالاسم
  - تحديث البيانات من Firebase
  - عرض جميع النتائج في جدول
- **الإجراءات:**
  - عرض تفاصيل كل نتيجة
  - حذف النتائج
  - تصدير البيانات إلى CSV

## 🗄️ قاعدة البيانات (Firebase Firestore)

### المجموعات (Collections):

#### 1. **questions** - الأسئلة
```json
{
  "id": "auto-generated",
  "text": "نص السؤال",
  "type": "mcq" | "tf" | "text",
  "time": 30,
  "options": ["الخيار أ", "الخيار ب", "الخيار ج"],
  "correct": 0,
  "createdAt": timestamp,
  "examId": "default"
}
```

#### 2. **exam_results** - نتائج الامتحانات
```json
{
  "id": "auto-generated",
  "studentName": "اسم الطالب",
  "score": 85,
  "correct": 17,
  "wrong": 3,
  "total": 20,
  "totalTime": 1200,
  "answers": [
    {
      "questionId": "...",
      "userAnswer": 2,
      "isCorrect": true
    }
  ],
  "timestamp": timestamp,
  "examId": "default"
}
```

## 🚀 كيفية الاستخدام

### الخطوة 1️⃣: إضافة الأسئلة
1. اذهب إلى `questions.html`
2. اختر نوع السؤال (MCQ / T&F / Text)
3. أدخل نص السؤال
4. حدد الإجابة الصحيحة
5. عيّن الوقت للسؤال
6. اضغط "إضافة السؤال"
7. سيتم الحفظ تلقائياً في Firebase

### الخطوة 2️⃣: إجراء الامتحان
1. اذهب إلى `exam.html`
2. أدخل اسمك الثلاثي
3. اضغط "ابدأ الامتحان"
4. أجب على الأسئلة (يختفي السؤال تلقائياً عند انتهاء الوقت)
5. انتقل بين الأسئلة أو انقر على نقطة للانتقال المباشر
6. انهِ الامتحان
7. سترى النتائج الفورية
8. ستُحفظ النتيجة تلقائياً في Firebase

### الخطوة 3️⃣: عرض النتائج
1. اذهب إلى `dashboard.html`
2. شاهد الإحصائيات العامة
3. ابحث عن طالب أو استعرض جميع النتائج
4. اضغط "التفاصيل" لرؤية تفاصيل النتيجة
5. صدّر النتائج إلى CSV إذا أردت
